# Base de datos

- SQL - Structured Query Languaje o Lenguaje de consultas estructuradas
- No es un lenguaje de programación
- SQL es un legunaje de consultas

## Conceptos fundamentales
- Entidad: respresenta un objeto del mundo real que tiene en el sistema ('Cliente)
- Tabla: forma en la que se almacena la información de una entidad en la base de datos
- Filas(registros): corresponde a una instancia concreta de la entidad ('cliente en especifico')
- Columnas(atributos): define una caracteristicas de la entidad ('nombre, direccion, precio)
- Llave(clave) primaria(PK): Identificador unico para cada fila en una tabla
- Llave(clave) foranea(FK): Relacion entre tablas, enlazando con la llave primaria de otra tabla


DBMS: DataBase Management System
